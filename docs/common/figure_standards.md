# Figure Standards

All manuscript figures should use the plotting conventions defined in the
active analysis report and shared report helpers.

## Design Requirements

* Use a white background and restrained, colorblind-accessible colors.
* Use Arial typography, black axis text, and minimal visual decoration.
* Use concise titles and subtitles; place methodological detail in captions.
* Label axes with units and use comma-separated large numbers.
* Use direct annotations only when they communicate a key result.
* Show uncertainty intervals clearly and state their type in the caption.
* Distinguish observed and forecast periods using both color and line style or
  a boundary marker.
* Avoid relying on color alone to communicate comparisons.

## Export Requirements

Every final figure must be exported in:

* Vector PDF
* LZW-compressed TIFF at 600 dpi
* PNG at 300 dpi for review

Use wider paper-friendly grids when a figure has many panels. For example, use
a 3 by 4 layout for 12-panel figures rather than a tall 2 by 6 layout. Text
must remain legible at final publication size.

## Scientific Requirements

* Do not imply statistical significance through visual styling alone.
* Identify whether intervals are GBD uncertainty intervals, confidence
  intervals, or prediction intervals.
* State when forecast intervals exclude GBD estimate uncertainty.
* Use consistent outcome ordering across related figures.
* Inspect every exported figure before reporting completion.
